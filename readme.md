# glados-pro

## About 
基于react->Node->技术栈对glodas进行重构,仅仅是一次实践。

## 用到的技术栈
 + React 构建Web组件
 + React-router 客户端路由及ssr路由
 + redux 管理应用state
 + react服务端渲染 实现同构
 + node/express 提供服务
 + webpack 构建
 + webpack-isomorphic-tools 处理ssr静态资源
 + JWTs 前后端分离下的token验证
 + redux-devtools 开发者工具
